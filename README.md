# prefix
CSS auto-prefix

## install

    npm install @pakastin/prefix
    
## oldskool
```html
<script src="https://pakastin.github.io/prefix/prefix.min.js"></script>
```

## usage

```js
import prefix from '@pakastin/prefix';

const transform = prefix('transform');

el.style[transform] = 'translate(100px, 0)';
```
